# Nuxt.js + Netlify Functions を TypeScript で構築する

## セットアップ

```bash
# インストール
$ yarn install

# Nuxt.js開発サーバ起動(localhost:3000)
$ yarn dev

# netlify-lambda開発サーバ起動(localhost:9000)
$ yarn dev:functions

# ビルド
$ yarn build
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
